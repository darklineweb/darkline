## 🚀 Getting Started

### 1️⃣ Upload Files to GitHub

* Create a repository
* Upload all project files

---

### 2️⃣ Enable GitHub Pages

Go to:

```
Settings → Pages
```

Select:

* Branch: `main`
* Folder: `/root`

---

### 3️⃣ Access Your Website

```
https://yourusername.github.io/repository-name/
```

---

## 🎬 Add Your Own Videos

1. Upload videos inside:

```
assets/videos/
```

2. Edit `app.js`:

```javascript
{
  title: "Your Video Title",
  file: "assets/videos/yourvideo.mp4"
}
```
